# go-on-metrics

Dashboard de métricas e análise de churn para Go On Outdoor.

## Tecnologias

- React + TypeScript + Vite
- Tailwind CSS
- Recharts para visualizações
- Excel.js para processamento de planilhas

## Funcionalidades

- Dashboard de métricas de churn
- Análise de coorte
- Gerenciamento de atletas
- Upload de planilhas Excel
- Visualizações interativas
